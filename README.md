# Barbell Path Tracking App

This Android app is designed to track the path of a weightlifting barbell using object detection powered by YOLOv8n. The app processes pre-recorded videos and overlays a tracking line to visualize the barbell's movement.

## Features (Planned)
- Manual selection of the barbell in a video
- YOLOv8n-based object detection for accurate tracking
- Visualization of the bar path
- Additional metrics such as rep count and duration (future updates)

## Status
This project is currently in development. Expect frequent updates and improvements.

## Requirements
- Android Studio
- Kotlin
- PyTorch or ONNX for running YOLOv8n models
